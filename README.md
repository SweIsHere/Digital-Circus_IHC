## Cómo usar el APK

Para instalar y probar la aplicación en un headset VR, sigue estos pasos:

1. **Headset compatible**: necesitas un Meta Quest 2, Meta Quest Pro o Meta Quest 3 (Meta Quest).
2. **Meta Quest Developer Hub (MQDH)**: instala [Meta Quest Developer Hub](https://developer.oculus.com/meta-quest-developer-hub/) en tu PC.
3. **Modo desarrollador**: activa el modo desarrollador en tu cuenta y headset desde la app de Meta Horizon (Meta Quest) en tu celular.
4. **Conectar el headset a la PC**: conecta el Quest a la PC vía USB (o Air Link) y autoriza el acceso a datos desde el visor cuando se solicite.
5. **Subir el APK**: en Meta Quest Developer Hub, ve a la sección de dispositivos, selecciona tu headset conectado y sube el archivo `APK/Puzzletadc.apk`.
6. **Instalar y ejecutar**: una vez subido, instala el APK desde el Hub y busca la aplicación en la sección "Apps no conocidas" dentro del visor para iniciarla.
